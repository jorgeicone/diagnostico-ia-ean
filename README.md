# diagnostico-ia-ean

Diagnóstico de apertura · Taller **Inteligencia Artificial Aplicada** (4 horas) — Universidad Ean.

- 20 preguntas de nivel en 5 dimensiones espejo de los seis ecosistemas del taller:
  Evidencia (Perplexity) · Contexto (Google) · Construcción (Claude) · Operación y entrega (Canva + Copilot) · Criterio y costo (GLM-5.2).
- 5 preguntas abiertas que cosechan el caso real que cada participante construirá en sesión.
- Backend: Supabase (`diagnostico_ia`, curso `ia-aplicada-ean`). Panel del facilitador + dashboard público (`sintesis.html`).
